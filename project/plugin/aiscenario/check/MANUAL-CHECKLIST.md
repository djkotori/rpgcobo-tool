# AI Scenario Importer — 手動スモークチェック

自動チェック (`check-aiscenario.ps1`) は Squirrel コンパイル・RPG-Cobo UI までは検証しません。
リリース前または UI 変更後は、以下を **M001 等のテストマップ** で確認してください。

## 起動・メニュー

- [ ] RPG-Cobo 起動時に `[aiscenario] plugin loaded` がコンソールに出る（起動不能になっていない）
- [ ] **編集 → シナリオメーカー読み込み** が表示される
- [ ] サブメニューは **JSON を貼り付け** / **JSON ファイルを選択** の 2 つのみ（独立した「履歴」項目はない）

## JSON 貼り付けダイアログ

- [ ] ダイアログ上部に **JSON 貼り付け / 履歴** タブがある
- [ ] **JSON 貼り付け** タブ: クリップボード JSON 編集、プレビューのみ、取り込みが動く
- [ ] **履歴** タブ: 日時 | 名前 | 先頭内容 | 分岐 の一覧が見える（空ならプレースホルダ表示）
- [ ] 履歴タブで項目選択 → **取り込む** で再 import できる

## 取り込み本体

- [ ] `sample/villager.json` を貼り付け → M001 に村人追加 → 会話できる
- [ ] `sample/custom.json` を貼り付け → custom イベント追加 → 分岐会話が動く
- [ ] 対象マップを**編集中**ならギズモ即表示・**右クリックメニュー**が出る
- [ ] 取り込んだイベントを **Delete** してもクラッシュしない（ギズモ無しイベント含む）

## 履歴ファイル

- [ ] import 成功後 `tmp/aiscenario-history.json` が更新される
- [ ] 31 回以上 import しても履歴は 30 件以内

## ファイル選択

- [ ] JSON ファイル選択 → プレビュー → import 成功
- [ ] 成功後履歴タブに同じ内容が増える
